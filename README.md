## 🚀 Featured Project — BillSmart

<div align="center">

> **AI-powered receipt scanner & expense tracker — built for India.**

</div>

BillSmart solves a problem 500 million UPI users face daily: where did the money actually go? Point your camera at a receipt, and BillSmart handles the rest.

| Feature | Details |
|--------|---------|
| 📸 **Receipt Scanning** | EasyOCR extracts line items and totals from any receipt photo |
| 🏷️ **Smart Categorization** | Auto-tags expenses (Food, Transport, Utilities, etc.) |
| 📊 **Live Dashboard** | Streamlit-powered real-time spending analytics |
| 🇮🇳 **Built for India** | Handles Indian receipt formats, UPI references, and regional text |

**Stack:** Python · EasyOCR · Pandas · Streamlit · SQLite
